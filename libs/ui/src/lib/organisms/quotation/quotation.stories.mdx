# Quotation

The Quotation component can be used to show personal quotes about Vanguard history, performance and other related information.

<Meta
  title="Organisms/Quotation"
  component={QuotationComponent}
  parameters={{
    design: {
      type: 'figma',
      url: 'https://www.figma.com/file/8HCSreWVbMUgCpsJugJsDf/Public-(Copy)?node-id=7901%3A23545',
    },
  }}
  argTypes={{
    quote: {
      control: { type: 'text' },
    },
    image: {
      control: { type: 'select' },
    },
    author: {
      control: { type: 'text' },
    },
    occupation: {
      control: { type: 'text' },
    },
    altText: {
      control: { type: 'text' },
    },
  }}
/>
