# Banner Hero

The Banner Hero displays a full page width image and/or [Pattern](?path=/docs/atoms-pattern) with accompanying title and optional body copy and additional CTA Link.

## Variants

### With Image

Display an image across the full page width.

<Canvas>
  <Story
    name="with Image"
    args={{
      title: 'Vanguard. Zusammen mehr vermögen',

      image: 'images/background-large.jpg',
      cta: {
        text: 'Jetzt Anlagevorschlag erstellen',
        routerLink: 'home',
      },
    }}
    argTypes={{
      showPattern: { table: { disable: true } },
    }}

    {Template.bind({})}
  </Story>
</Canvas>
